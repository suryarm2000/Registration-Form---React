# Event Registration Form (React Practice)

This is a React practice project I built during my frontend developer internship at Sharpsys.

I was given tasks step by step by my mentor, and I implemented them in React:
- showing form values
- controlled components
- styling to match the given design
- responsiveness for mobile (in progress)

## What I used
- React
- JavaScript
- HTML
- CSS

## What this project does
- Collects user details using a form
- Uses controlled inputs in React
- Shows entered values in the UI (example: name / date of birth)
- Includes sections like participant details and payment details
- Styled to match a reference layout
- Responsive layout for mobile screens

## What I learned
- React component structure
- `useState` and controlled inputs
- Handling form input changes
- Basic validation / input restrictions
- CSS layout using Flexbox / Grid
- Media queries for responsive design
